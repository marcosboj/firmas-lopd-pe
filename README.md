# Proyecto de Gestión de Firmas (ECODES)

Este es el proyecto principal para la gestión de firmas desarrollado para **ECODES** (Fundación Ecología y Desarrollo).

## Tecnologías Principales

*   [Next.js 14/15](https://nextjs.org/) (App Router)
*   [React](https://reactjs.org/)
*   [TypeScript](https://www.typescriptlang.org/)
*   [Tailwind CSS](https://tailwindcss.com/)
*   [Lucide React](https://lucide.dev/) (Iconos)

## Estructura del Proyecto

*   `/app`: Rutas de la aplicación (Next.js App Router).
*   `/components`: Componentes reutilizables de la interfaz de usuario.
*   `/hooks`: Custom hooks de React.
*   `/lib`: Funciones de utilidad (ej. integraciones, utilidades CSS como `clsx` y `tailwind-merge`).

## Desarrollo

Para iniciar el servidor de desarrollo local:

```bash
npm run dev
# o
yarn dev
# o
pnpm dev
# o
bun dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.
